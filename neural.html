# Activation Functions

In Aegironox, we primarily use ReLU (Rectified Linear Unit) for hidden layers to prevent the vanishing gradient problem. For the output layer in classification, Softmax is used: \(\sigma(z)_i = \frac{e^{z_i}}{\sum e^{z_j}}\).

# Backpropagation

The core of our learning process is Gradient Descent. We calculate the derivative of the loss function with respect to each weight using the chain rule. This allows the network to "learn" from its mistakes in every epoch.

# Convolutional Neural Nets

For Robotics vision, CNNs are essential. They use filters to detect edges, textures, and eventually objects in the LiDAR or camera feed, enabling the autonomous rover to navigate without collisions.
